### Construction Methods

#### 1. Spray-Freeze Method

```
Process:
1. Pump seawater through spray nozzles
2. Freeze in cold air (-30¬∞C winter)
3. Build layers (1m/day in peak winter)
4. Reinforce with ice fibers
5. Shape with heated molds
```

**Output:** 10,000 m¬≥/day per station

#### 2. Ice Block Assembly

```
Process:
1. Harvest natural ice (icebergs, glaciers)
2. Cut into standardized blocks (10√ó5√ó3m)
3. Transport to construction site
4. Fuse with water spray (freezes as bond)
5. Reinforce with internal cables
```

**Output:** 5,000 blocks/day

#### 3. 3D Ice Printing

```
Process:
1. Mix ice powder + water binder
2. Extrude through heated nozzle
3. Freeze layer-by-layer
4. Embed reinforcement fibers
5. Smooth with heated tools
```

**Output:** 1,000 m¬≥/day (precision structures)
