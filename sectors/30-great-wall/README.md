## üèóÔ∏è Construction Methods

### 1. Rammed Earth (Â§ØÂúü)

**Process:**
```
1. Build wooden frames (formwork)
2. Layer soil, gravel, lime (15-20cm)
3. Compact with wooden rammers (100+ workers)
4. Repeat until desired height
5. Remove formwork, cure in sun
```

**Material Composition:**
| Component | Percentage | Purpose |
|-----------|------------|---------|
| **Loam soil** | 60% | Base material |
| **Gravel** | 20% | Drainage, stability |
| **Lime** | 10% | Binding agent |
| **Sand** | 8% | Texture |
| **Sticky rice** | 2% | Organic binder (Ming) |

**Strength:** 3-5 MPa (comparable to modern concrete: 20-40 MPa)

### 2. Stone Masonry (Ming Dynasty)

**Process:**
```
1. Quarry local granite/limestone
2. Cut blocks (standardized sizes)
3. Transport via sledge, roller, human chain
4. Lay with lime mortar
5. Fill core with rubble, earth
```

**Block Sizes:**
| Location | Length | Width | Height | Weight |
|----------|--------|-------|--------|--------|
| Badaling | 1.5m | 0.5m | 0.3m | 600kg |
| Mutianyu | 1.2m | 0.4m | 0.25m | 400kg |
| Jinshanling | 1.0m | 0.4m | 0.3m | 350kg |

### 3. Brick Production (Ming)

**Kiln Process:**
